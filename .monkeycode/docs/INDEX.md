# CET 词汇文档

CET 词汇是一个纯前端 PWA 闪卡应用，支持 CET-4/CET-6 双词库刷词、生词本复习、每日打卡。本文档面向开发者，涵盖架构设计、数据接口和开发规范。

**快速链接**: [架构](./ARCHITECTURE.md) | [接口](./INTERFACES.md) | [开发者指南](./DEVELOPER_GUIDE.md)

---

## 核心文档

### [架构](./ARCHITECTURE.md)
系统设计、技术栈、子系统划分、架构图和新用户注册流程。从这里开始了解系统全貌。

### [接口](./INTERFACES.md)
IndexedDB 数据契约、JSON 词库格式、DOM 事件系统和 CSS 变量接口。集成或修改系统时的完整参考。

### [开发者指南](./DEVELOPER_GUIDE.md)
环境搭建、编码规范、常见任务（添加词库、添加设置项、部署上线）和边界情况处理。

## 快速参考

### 启动开发服务器

```bash
python3 -m http.server 8000
```

### 关键文件

| 文件 | 作用 | 行数 |
|------|------|------|
| `index.html` | 完整 PWA 应用 | ~1100 |
| `words_cet4.json` | CET-4 词库 | 2607 词 |
| `words_cet6.json` | CET-6 词库 | 548 词 |

### 关键变量

| 变量 | 类型 | 说明 |
|------|------|------|
| `BANKS` | Object | 词库注册表 `{key: {name, file}}` |
| `currentBank` | string | 当前词库 key |
| `words[]` | Array | 当前词库全部单词 |
| `data` | Object | 当前用户+词库的 StudyData |
| `settings` | Object | 当前用户+词库的 UserSettings |
| `uid` | string | 当前用户唯一标识 |
| `studyMode` | string | 当前刷词模式 `order|shuffle|bookmark` |
| `index` | number | 当前刷词位置 |
